Scaffolding in Tableau Prep¶

I found a dataset from kaggle containing about 9,000 games from around Europe which offered a lot of detailed information about players and the events that happen during games.

In the dashboard above, one can see red boxes when Hazard didn't perform the event type selected from the filter. This means the average times he performed the event is 0. However, in the original data set, only events that actually occured will have a record in the dataset. How was I able to show a red box with 0 when a record for that doesn't exist? This is where scaffolding comes into play.

My scaffolding process¶

Scaffolding refers to the manipulation of data so that a visualization can be made. I did my data scaffolding in Tableau Prep. This allowed for joining data on the fly and the ability quickly check results. To create the visualization I wanted, where an average of 0 could be shown instead of nothing, I needed to make every combination of game, event type, and field location. That way, even if Hazard didn't score a goal in a game, there would be a record I could use to display a 0 in Tableau. To make every combination of game, event, and field location, I simply made a separate self-join using a column of 1's as the join field.

Each Combination¶

Above we can see a screenshot of my Tableau Prep flow. The box highlighted in a blue border is the join between all games and all event types. The two summation symbols before this step labeled "Games" and "Events" are aggregation steps to select distinct games and distinct events while also creating a column of 1's. The window at the bottom shows the join results. After using 1 as the join clause, each event will be matched with each game.

This same process is done for the locations on the field to create each combination of event, game, and location. The resulting combinations can be seen below.

I also created a calculated field to identify which records actually occured in the game. For the data of all combinations, the value for 'occur all' is 0.

The next step is to join the data of all combinations back to the original data. The original data also has a calculated field to keep track of events that actually occurred. This takes the value of 1. The result of the join can be seen below. The resulting data set has a record for each game, event type, and. field location for Eden Hazard. This allowed me to visualize data that was not initially present in the original data set. Sometimes, people will simply pad a data set in excel with the missing records, but Tableau Prep offers a great option to scaffold data and guarantee all data points are covered.
