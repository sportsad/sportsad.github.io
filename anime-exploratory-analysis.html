We will continue our exploration with some outlier analysis.

The boxplots above have upper and lower fences to determine outliers. We can calculate these using the interquartile range and get a closer analysis of the outliers. The IQR is found by subtracting the first quartile from the third quartile. The first quartile is the median of the bottom half of the data of which 25% of the data fall below. The thrid quartile is the median of the top half of the data. 75% of the data falls below this. The IQR captures the middle 50% of the data.

When we multiply the IQR by 1.5 and subtract it from the from the first quartile, we obtain the lower whisker or fence. Data points below this are considered mild outliers. In the same manner, adding 1.5*IQR to the third quartile creates the upper fence.

We can identify extreme outliers by multiplying 3*IQR and adding/subtracting from the two quartiles to create two outer fences. These aren't shown in the boxplot, but can be useful to identify outliers are extreme data points.

In [61]:
q1 = tv_anime_long.groupby('genre_list')['rating'].quantile(q=0.25)
q3 = tv_anime_long.groupby('genre_list')['rating'].quantile(q=0.75)
iqr = q3-q1
upperFence = q3 + 1.5*iqr
lowerFence = q1 - 1.5*iqr
outerUpperFence = q3 + 3*iqr
outerLowerFence = q1 - 3*iqr

Here, I define two functions to identify outliers, one for mild outliers one for extreme outliers. I also added the detail of making outliers on the lower end differentiated from outliers on the upper end. Later on, we'll be able to look into shows that were rated abnormally high and low.

In [62]:
def outliers(group):
    genre = group.genre_list
    return 1 if (group.rating > upperFence.loc[genre]) else -1 if (group.rating < lowerFence.loc[genre]) else 0

def extreme_outliers(group):
    genre = group.genre_list
    return 1 if (group.rating > outerUpperFence.loc[genre]) else -1 if (group.rating < outerLowerFence.loc[genre]) else 0

We can see from the comparison of interquartile ranges, the two batches of data, tv and movies, have differing spread. With different spreads of data, it is more difficult to compare groups by a measure of center; the median is simply a mere data point. To help see the differences in spread and averages of the data, we can use a spread vs. level plot. I compute the medians for the TV data and Movie data, and plot the log of the medians vs. the log of the interquartile ranges. This plot can help determine a transformation to make equal spreads for easier comparison.
